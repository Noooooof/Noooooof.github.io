整理备份文件的时候顺手记录下常用软件们：

  • Everything：小巧而迅速的文件搜索软件，根据名称定位，会记忆上一次对某一关键词的搜索结果——可以用来快速启动其他程序。
  • CCleaner：清理垃圾、清理注册表、管理自启动进程和插件etc。免费版够用了，想买付费版等一等黑色星期五，会有相当优惠的价格。
  • ESET：习惯了，存在感不高的杀毒软件。不过有windows defender的情况下杀毒软件有什么用吗？
  • 7-zip：解压软件，你永远可以相信7z。
  • IObit Uninstaller：卸载软件，同样免费版够用。
  • Chrome：界面干净操作舒适，扩展程序非常多，没有迁移其他浏览器的意愿。
    • aria2：辅助下载大文件，Chrome自己不带断点续传，下Unity的时候反复99%挂掉好绝望。（Chrome91版本中似乎挂了）
  • Typora：优雅的markdown文本编辑器，所见即所得。据说Mac上面显示效果更好些。
  • eDiary：很棒的本地日记本，反应迅速，界面简单，可以设置密码，有一个联网获取天气的小插件。
  • Obsidian：适用于管理一个文件库的笔记软件，之前有介绍。
  • Sublime Text 3：文本编辑器。高中沉迷于自定义，现在沦为了python启动器x
  • Notion：全平台同步，数据库是个有趣且方便的设定，拿来整理资料和记录日程。
  • OneNote：全平台同步，手写支持很不错，界面美观格式齐全，笔记软件的无冕之王。
  • Adobe Acrobat Pro DC：电脑上用了一堆pdf阅读器还是Acrobat最香。
  • PotPlayer：视频播放器，疫情在家为了二倍速看录播换的，蛮好用。
  • v2rayN：You-know-what : )
